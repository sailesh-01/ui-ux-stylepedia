# UI/UX Stylepedia

🎨 Interactive compendium of 50+ UI/UX design styles – explore, feel, and inspect Glassmorphism, Neumorphism, Brutalism, Cyberpunk, and many more. Live theming, CSS snippets, AI prompts, and a premium‑grade dashboard—all in pure HTML/CSS/JS.

## Features
- Full‑screen style grid with category filters (Classic Foundations, Modern & Popular, Experimental, Platform & Industry).
- Live theme preview – click a card and the entire page morphs to that style.
- Copy code & stitching prompts for easy reuse.
- Master footer with quick links that filter the grid or open the About modal.
- Responsive design – works beautifully on desktop and mobile.
- Zero dependencies – just vanilla web technologies.

## Getting Started
Open `index.html` in a browser or click start.bat to explore the UI. No build step required.
